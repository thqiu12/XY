# 株式会社XY 公式サイト

教育機関向けのシステム開発・DX支援を行う株式会社XYのコーポレートサイト。
銀行の法人口座開設審査に対応することを最優先要件として構築。

## 構成（静的サイト・ビルド不要）

| ファイル | 内容 |
|---|---|
| `index.html` | トップ（ヒーロー / 課題 / サービス / 強み / 代表 / 導入事例 / CTA） |
| `company.html` | 会社概要（登記事項に準拠した全項目テーブル） |
| `contact.html` | お問い合わせ |
| `privacy.html` | プライバシーポリシー |
| `css/style.css` | 全ページ共通スタイル |
| `assets/` | ロゴ・favicon |
| `CNAME` / `robots.txt` / `sitemap.xml` | デプロイ・SEO 用 |

外部依存は CDN のみ（Google Fonts: Noto Sans JP、Phosphor Icons）。Node/ビルド不要。

## ローカル確認

```bash
python3 -m http.server 8000
# http://localhost:8000 を開く
```

## デプロイ（独自ドメイン xy-cloud.org）

### GitHub Pages の場合
1. このリポジトリを push
2. Settings → Pages → Source を `main` / `(root)` に設定
3. `CNAME`（xy-cloud.org）が含まれているため、ドメイン側で以下の DNS を設定:
   - `A` レコード → `185.199.108.153` / `185.199.109.153` / `185.199.110.153` / `185.199.111.153`
   - もしくは `www` を `CNAME` → `<user>.github.io`
4. Pages の「Enforce HTTPS」を有効化

### レンタルサーバー / Vercel / Netlify の場合
ファイル一式をそのまま公開ディレクトリに配置するだけで動作します。

## 内容の更新ポイント

- **導入事例**（`index.html` の `#cases`）：現在は匿名・中性の枠組み。実在事例の詳細・数値が確定したら差し替え。
- **代表写真**（`index.html` の `.rep__avatar`）：現在は姓のイニシャル表示。実写真に差し替え可能。
- **受付時間 / 営業時間**（`contact.html`）：実際の運用に合わせて調整。
